## 2. Main writeup

Das Ziel ist es, rigorosen partiellen Fortschritt (rigorous partial progress) fΟΦr die Frage zu erzielen, ob $S = \sum_{n \ge 1} p_n / 2^n$ irrational ist[cite: 1]. Wir nutzen den im Dossier beschriebenen Squeeze-Mechanismus und reduzieren das Problem auf eine explizit quantifizierte, sehr schwache wachstumstheoretische Bedingung ΟΦber aufeinanderfolgende PrimzahllΟΦcken (Consecutive Primes in Arithmetic Progression)[cite: 1].

**Unbedingte Vorarbeit (Beweis von CLAIM-1):**
Es ist eine elementare Tatsache, dass fΟΦr die $k$-te Primzahl $p_k < k^2$ fΟΦr alle $k \ge 2$ gilt. Wir definieren das Tail-Gewicht $u_n = \sum_{k \ge 1} p_{n+k}/2^k$[cite: 1]. Wir kΟΕnnen $u_n$ strikt nach oben abschΟΛtzen:
$$u_n = \sum_{j=1}^{\infty} \frac{p_{n+j}}{2^j} < \sum_{j=1}^{\infty} \frac{(n+j)^2}{2^j}$$
Durch Ausmultiplizieren von $(n+j)^2 = n^2 + 2nj + j^2$ und Nutzung der Standardreihen $\sum_{j=1}^\infty 2^{-j} = 1$, $\sum_{j=1}^\infty j 2^{-j} = 2$ und $\sum_{j=1}^\infty j^2 2^{-j} = 6$ erhalten wir:
$$u_n < n^2(1) + 2n(2) + 6 = n^2 + 4n + 6$$
Da $\delta_n = u_n - p_{n+1}$ und $p_{n+1} > 0$[cite: 1], folgt strikt $\delta_n < n^2 + 4n + 6$.

**Die SEGR-Hypothese (Sub-Exponential Equal Gap Runs):**
Wir formulieren folgende exakte Hypothese ΟΦber die Verteilung von PrimzahllΟΦcken:
*Es existiert eine gerade ganze Zahl $c > 0$ und ein $\alpha \in (0, 1/2)$, sodass fΟΦr unendlich viele LΟΛngen $J$ ein Startindex $N = N(J)$ existiert, bei dem $J$ aufeinanderfolgende PrimzahllΟΦcken konstant sind ($g_{N+1} = g_{N+2} = \dots = g_{N+J} = c$), wobei das Auftreten frΟΦh genug stattfindet, nΟΛmlich $N(J) \le 2^{\alpha J}$, und gleichzeitig exakte AuslΟΕschung vermieden wird: $\delta_{N+J} \ne c$.*

**Bedingtes Theorem (Beweis von CLAIM-3):**
Angenommen, $S$ ist rational. Dann existieren ganze Zahlen $a, s \ge 0$ und eine ungerade Zahl $b > 0$, sodass $S = a / (2^s b)$[cite: 1]. Daraus folgt zwingend, dass $b \delta_n \in \mathbb{Z}$ fΟΦr alle $n \ge s$[cite: 1].

Unter der SEGR-Hypothese wΟΛhlen wir ein $J$, das groΟü genug ist, sodass $J > s$ und die Ungleichung $b ( (2^{\alpha J} + J)^2 + 4(2^{\alpha J} + J) + 6 + c ) < 2^J$ gilt. Da $\alpha < 1/2$ ist, wΟΛchst die linke Seite mit $\mathcal{O}(2^{2\alpha J})$, was asymptotisch streng von $2^J$ dominiert wird, sodass diese Ungleichung fΟΦr hinreichend groΟüe $J$ zwingend erfΟΦllt ist.

FΟΦr dieses $J$ und den zugehΟΕrigen Index $n = N(J)$ liefert der Squeeze-Mechanismus die IdentitΟΛt:
$$\delta_n - c = 2^{-J}(\delta_{n+J} - c)$$
Multipliziert mit dem Nenner $b$:
$$b \delta_n - b c = b 2^{-J} (\delta_{n+J} - c)$$
Da $n = N(J) \ge J > s$, muss die linke Seite eine ganze Zahl sein[cite: 1]. Die rechte Seite muss folglich ebenfalls integer sein. Wir prΟΦfen den Betrag der rechten Seite:
$$0 < | b 2^{-J} (\delta_{n+J} - c) | \le b 2^{-J} (\delta_{n+J} + c)$$
Die Striktheit grΟΕΟüer Null folgt direkt aus der Bedingung $\delta_{n+J} \ne c$ der Hypothese. Wir wenden CLAIM-1 auf $\delta_{n+J}$ an:
$$b 2^{-J} (\delta_{n+J} + c) < b 2^{-J} ( (n+J)^2 + 4(n+J) + 6 + c )$$
Da $n = N(J) \le 2^{\alpha J}$, ist dieser Ausdruck nach Wahl von $J$ strikt kleiner als $1$. 
Wir haben somit eine ganze Zahl konstruiert, die strikt zwischen $0$ und $1$ liegt. Dies ist ein Widerspruch. Folglich muss $S$ irrational sein.
